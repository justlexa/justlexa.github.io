Programming languages

JavaScript

JavaScript is a lightweight interpreted programming language used mostly in webpages.

Python

Python is used alot in things like research and machine learning, and is an interpreted, object-oriented programming language.

C#

As a general purpose language, C# is used for lots of different things from mobile apps, desktop apps to website, etc. It is an object-oriented programming language.

C++

C++ is one of the most widely-sued programming languages in game development, having been used to create lots of popular games such as World of Warcraft and Counter-strike. It is a light-weight abstraction programming language.

Lua

Lua is a glue language, developed to extend and integrate programs. It is widely used as a scripting language by game programmers.
